POWER APPS
    TIPOS DE APLICACIONES QUE SE PUEDEN CREAR
        Lienzo
            aplicaciones personalizadas y diseñadas para tareas especificas
            interactuan con datos de multiples origenes
            se pueden utilizar formulas de Powe FX
            poseen formulas muy similares a las de excel
            son faciles de crear a partir de datos ya existentes
            el creador tiene control sobre el origen de los datos
            orientadas al cliente o usuario
            
            CARACTERISTICAS DIFERENCIADORAS CON RESPECTO A LAS APLICACIONES DE MODELOS
                No basada en dataverse
                Centrada en tarea o pantalla
                interfaz de usuario personalizada
                integracion de dispositivos
                se inserta con facilidad
            
        Modelos
            siempre se crean a partir de datos en Microsoft dataverse
            utilizan un modelo que prioriza los datos
            se centra en agregar componentes como formularios, vistas, gráficos y paneles de información
            se utilizan como aplicaciones de gestion
            orientadas al administrador o director, para actividades como realizar seguimientos con datos
            
            CARACTERISTICAS DIFERENCIADORAS CON RESPECTO A LAS APLICACIONES DE            
                Basada en dataverse
                centrada en procesos u operaciones administrativas
                interfaz de usuario con capacidades de respuesta/ uninforme
                personalizacion de usuario
                navegacion de relaciones de datos
                recote de seguridad de la interfaz de usuario
                
        las aplicaciones de modelos sirven para gestionar y organizar a las aplicaciones de lienzo, y mientras que las aplicaciones de lienzo sirven para satisfacer las necesidades del cliente, las de modelos, se encargar de producir informes para que los administradores tomen decisiones en consecuencia de los resultados que se presenten
